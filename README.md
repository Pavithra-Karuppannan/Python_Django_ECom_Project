<h1 style="color:blue;">ECOMMERCE PROJECT</h1>

Developed By : Pavithra

Description: A full-stack E-Commerce web application built using Python and Django. The project includes user authentication, product listings, shopping cart functionality, favorites/wishlist, and an admin dashboard for managing products and categories. Designed with responsive frontend templates and backend database integration to provide a seamless online shopping experience.
